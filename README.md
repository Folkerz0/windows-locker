# Ultra-Fast System Locker

Windows lock screen that starts instantly on boot.

## Download

1. Go to [Releases](../../releases) or [AppVeyor builds](https://ci.appveyor.com/project/YOUR_USERNAME/REPO_NAME/build/artifacts)
2. Download `LockScreen.exe`
3. Run once to create startup shortcut
4. Restart PC - locker appears instantly

## Features

- Starts in &lt;100ms
- Blocks Alt+Tab, Win key, Ctrl+Alt+Del
- No internet required
- 100% offline
- ~50KB executable

## Build

Auto-builds on every commit using AppVeyor CI.
